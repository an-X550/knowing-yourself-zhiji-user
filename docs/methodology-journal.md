# 如何写好日志——方法论

> 来源：问问大象日记系统教学
> 整理日期：2026-07-05

---

## 一、什么是"日志"

### 每日自我评估和诊断

一天的学习和生活结束，给自己进行一次成长评估：

- **有什么收获**：让自己开心、充实的，被他人善意帮助值得感谢的
- **思考**：为什么这些事情让自己开心——是因为自己喜欢、擅长，还是符合自己的志向和三观
- **改进**：如果再来一次，这么做能把这个事情做得更好，或避免陷入负面的状态

### 日志和传统应试评测的主要区别

| 维度 | 日志 | 传统应试评测 |
|------|------|------------|
| **关注目标** | 个人的六维成长 | 知识点或考点的掌握 |
| **评分方式** | 加分制（上不封顶） | 扣分制（满分100分） |
| **关注重点** | 积极变化，兴趣/优势/收获的挖掘 | 扣分点和错题 |
| **评判标准** | 自己的感受 | 他人和外界标准 |

本质上是**大学及未来职场游戏规则**和**各种应试游戏规则**不同：

- 社会及职场的竞争核心：能否扬长避短，而不是"想高分就不能有短板"
- 人生幸福因人而异

## 二、为什么写好日志能快速成长

类比传统应试学习中的"评测诊断"作用：

- 能写好日志并持续坚持的人，善于观察总结，提炼好的经验，吸取坏的教训
- 浑浑噩噩的人，当生活NPC

> **核心原理**：快速成长的关键在于能否围绕核心目标（六维图），持续高效地积累好的东西，抛弃糟粕。

日志扮演的角色类似传统学习中的评测诊断——能快速帮助自己抓住重点，查缺补漏，提炼好的经验，吸取坏的教训。

## 三、怎么写日志——完整六步法

### 第一步：回忆事实

**做什么**：快速回顾今天一天是怎么度过的，发生了哪些事情。

**关键要点**：这一步只回顾客观事实，先不要筛选或评价。

**回忆例子**：
- 我今天上午做了什么事，结果如何
- 下午又和谁互动，大概情况如何

### 第二步：筛选重点

**做什么**：从今天发生的事情里，筛选出能关联成长目标（六维图）的有收获的事情。

**六维图**（六大成长维度）：

| 维度 | 说明 | 例子 |
|------|------|------|
| **大学适应** | 游戏规则理解 | 今天上了大象第一节课，才知道原来好学历+成绩不等于好工作 |
| | 大学人际关系 | 今天又和室友发生了冲突，以前没住校过不知道该怎么处理 |
| | 大学课业学习 | 今天又是自学微积分的一天，没办法老师讲得太差了 |
| **自我认知** | 天赋 | 今天尝试了下编程实战，很快就能上手，也许我有点天赋啊 |
| | 兴趣 | 入职学生会宣传部一段时间，发现我还挺喜欢设计海报的 |
| | 价值观 | 今天刷到大象的撑伞侠视频，突然很被触动，我觉得"利他"能给我带来意义感 |
| **探索实际** | 实践、实习、接触真实世界 | |
| **能力发展** | 做事能力、与人协作能力 | |
| **校招技术** | 简历、面试、职业规划 | |
| **系统思考** | 分析方法、认知框架 | |

**注意事项**：
- 筛选印象深刻的1-3件事情就行
- 有些事情和核心目标没关系，但印象深刻的也记
- 不要为记而记

### 第三步：评估结果

**做什么**：根据自己对这些事情的真实感受，分类到不同的感受栏：

- 开心
- 充实
- 要感激的人/事

**注意事项**：
- 重点是正面的感受，加分制
- 不要过于关注负面的感受
- 即使关注负面感受，重点也是可能的正向意义（如全面认识自我以及如何改进）

### 第四步：洞察思考

**做什么**：尝试去多想一步——为什么这些事情让自己开心、收获、想要去感谢？背后的原因是什么？

**注意事项**：
- 不强求，有才写，没有就算了
- 基于自己的真实感受和洞察，不要照搬别人的观点

### 第五步：行为改进

**做什么**：

- **坚持好的行为**：自己的哪些行为习惯是好的，要坚持
- **改进不够好的行为**：哪些行为需要做出改变，让自己变得更好

**注意事项**：
- 改进这栏对应的是行为，要付出行动
- 如："我要去投递简历"
- 如果只是一些思路想法或情绪感受，写到对应心情栏中

### 第六步：分享讨论

**做什么**：
- 分享日志到群聊和朋友圈，和"同行人"交流讨论
- 围绕日志里提到的重点感受和思考，去相互**提问和交流**

## 四、写不好日记的六大典型问题

### 问题一：目标错误

- **表现**：没有围绕正确的目标来筛选和评估自己一天做的事情
- **举例**：关联的成长目标是"内卷道路目标"——绩点、证书、保研（而非六维成长目标）

### 问题二：没法对号入座

- **表现**：不知道自己经历的事情和六维目标之间的内在关联，无法对应
- **根因**：事情无法对应目标，本质是没有掌握知识框架——需要补课
- **类比**：某道题不会做，但不知道是对应哪个知识点——靠记口诀套公式做题的遗毒

### 问题三：只测不学，无事可写

- **表现**：没有积极投身实践和学习，接触外人和外部世界，导致无事可写
- **改进**：找不到合适的平台实践——先去接触人，不要着急一步求成

### 问题四：写不出太多有营养的收获

- **主要原因**：
  1. 没有用心去实习和做事，习惯敷衍应对
  2. 对外界和他人关注不足，缺乏好奇心，没有养成观察的习惯
  3. 对自己感受和情绪关注不足
- **提升方向**：
  1. 转变心态，重视自己的感受、情绪以及与他人的人际关系
  2. 重视心理学、哲学、社会学、政治、历史、项目管理等通识知识的学习
  3. 简易版：认真学习问问大象相关的入门课程（门槛低，针对性强）

### 问题五：系统思考不足

- **特征**：
  - 系统思考和分析的能力不足，没办法搞清楚各种现象背后的根本原因以及改进策略
  - 联想启发不足，不能触类旁通、举一反三
- **改进**：
  - 老学员需要补课 + 持续长期练习
  - 学习**系统思考入门课**
  - 掌握**手段目的分析法**——做好选择的前提能力
  - 学习**如何正确提问**，可以使用DeepSeek练习

### 问题六：闭门造车

- **特征**：自己埋头总结和思考，从不和他人交流，也不向他人学习
- **后果**：很容易陷入认知茧房，停滞不前
- **改进**：
  - 群里提问，搜索圈子里的精选
  - 积极交流，开放学习

## 五、日记系统的其他应用

- **AI助教**：针对性的个人成长评估报告
- **全员成长报告**：社团活跃度做标准的筛选条件
- **持续积累**：为月志（月度复盘）提供原始数据
